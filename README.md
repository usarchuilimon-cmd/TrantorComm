# SyncFlow

SyncFlow is a comprehensive communication and management dashboard application. It is designed to unify customer interactions, contact management, and marketing campaigns into a single, intuitive interface.

## Key Features

-   **Unified Dashboard**: High-level overview of performance metrics.
-   **Multi-Channel Conversations**: Manage messages from WhatsApp, Instagram, and Messenger in one place.
-   **Contact Management**: A built-in CRM to track contacts, tags, and VIP statuses.
-   **Campaign Manager**: Tools to schedule and track WhatsApp marketing campaigns.
-   **Live Assistant**: Always-on virtual assistant to help with tasks.
-   **Responsive Design**: Fully functional on desktop and mobile devices.

## Tech Stack

-   **Frontend**: React 19, TypeScript, Vite
-   **Styling**: Tailwind CSS
-   **Icons**: Lucide React
-   **Charts**: Recharts
-   **AI**: Google GenAI SDK

## Getting Started

### Prerequisites
-   Node.js (LTS recommended)

### Installation

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Environment Setup:**
    Set the `GEMINI_API_KEY` in `.env.local` to enable AI features.

3.  **Run Locally:**
    ```bash
    npm run dev
    ```
